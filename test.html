Regras do Site

Confimar

Verificar se a bolinha azul está no time correto depois aperte o botão confirmar para adiciona a vitória do time que ganhou a partida.

Resultado

Para saber o resultado das vitorias a qualquer momento aperte o botão resultado. O resultado aparece abaixo

Finalizar

Para finalizar a sessão e só apertar o botão finalizar e em seguida ver o time que ganhou abaixo. Logo depois apertar o botão restart para começar uma nova sessão.

Restart

Assim que você apertar o botão finalizar aperte o botão restart para iniciar uma nova contagem cancelando a anterior.

Apagar Seleção

O botão apagar seleção serve para apagar um resultado que você tenha colocado errado. Para que você possa apagar corretamente:

  1. Precisa verificar se a bolinha azul esta no time que você queira apagar.
  2. Tem que selecionar o resultado que você quer apagar apertando na setinha para baixo no quadrado em branco aonde estão armazenado os resultados.
  3. Caso você tenha apagado e a bolinha azul não esteja em cima do time que você queria apagar favor adicionar o time que você apagou errado novamente, para que a contagem não fique errada.
